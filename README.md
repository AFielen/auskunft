# DRK Selbstauskunft

Digitale Selbstauskunft für Vorstände, Geschäftsführer und Prokuristen im DRK.

## Features
- Geführter Wizard (6 Abschnitte + Unterschrift)
- Bedingte Felder (z.B. Betriebsrat-Fragen nur bei vorhandenem BR)
- Automatische Abweichungserkennung mit Pflichtbegründung
- Responsive / Mobile-First

## Entwicklung

```bash
npm install
npm run dev
```

## Tech-Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
